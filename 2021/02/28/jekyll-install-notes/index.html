Trouble shooting

Bump kramdown from 1.17.0 to 2.3.0

Gemfile.lock updated due to security warning, after which bundle exec jekyll serve will always fail

SSH
  1. Add the content of ~/.ssh/id_rsa.pub to GitHub -> Settings -> SSH and GPG keys -> New SSH Key
  2. Verify: ssh -T git@github.com
  3. Update [remote url] in .git/config
